Settings · GitHub connection

Token: create at github.com → Settings → Developer settings → Fine-grained tokens. Repository access: only this repo. Permissions: Contents → Read and write. Treat it like a password.

Passphrase: pick something long and memorable. The TV unlocks with the same passphrase added to the board URL after a hash, e.g. …/daily.html#k=YourPassphrase. The part after # is never sent over the internet. Change it any time, then republish and update the YoDeck URL.

Important: Safari and a Home Screen icon keep separate storage on iPhone. Enter and save these settings inside whichever one you actually use each day. If saving fails, check iPhone Settings → Apps → Safari → Block All Cookies is OFF, and avoid Private Browsing.
