Settings · GitHub connection

Token: create at github.com → Settings → Developer settings → Fine-grained tokens. Repository access: only this repo. Permissions: Contents → Read and write. Treat it like a password.

Passphrase: pick something long and memorable. The TV unlocks with the same passphrase in the board URL, e.g. …/daily.html#k=YourPassphrase. Change it any time, then republish and update the TV URL.

Important: browsers keep separate storage. Enter and save these settings inside whichever browser or icon you actually use each day. If saving fails, allow cookies/storage for this site.
